# AI チャットボット

## 要件
- OpenAI APIを利用した高度な自然言語処理が可能なチャットボットアプリケーション。
- リアルタイムレスポンスと履歴管理機能を実装。
- ログイン機能を実装


## 技術スタック
- React
- Node.js
- OpenAI API
- MongoDB

## 機能要件
- ユーザーがチャットボットと対話できるようにする。
- リアルタイムのチャットレスポンスを提供する。
- チャット履歴を保存し、表示する。

## データモデル
- ユーザー名
- チャット履歴
- チャット内容
- チャット日時

## チャット履歴の保存
- チャット履歴をMongoDBに保存する。
- チャット履歴を表示する。

## ログイン
- ユーザー名を入力する。
- ユーザー名を保存する。
- ユーザー名を表示する。
